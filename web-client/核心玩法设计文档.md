# 游戏核心玩法(P级别代表实现的优先级)
## 法宝系统(P0)
1. 法宝存放丹田里。
2. 法宝配置占人物的法宝点数，会影响出场的数量。
3. 法宝可以装备到丹田栏，丹田栏不会随着境界的增加而增加，金丹境界丹田栏解锁后只能通过加点来增加丹田栏的大小。
4. 丹田拼图玩法：
    1. 法宝的大小和形状都不一样，在丹田里也是对应大小的，越强大的法宝，对应的丹田越大，形状越奇特，增加“空间管理”的策略性。
5. 法宝召唤顺序与冷却玩法：
    1. 每回合只能召唤出场点数范围内的法宝，需要设计每种法宝的出场点数。
    2. 引入“行动点”机制，限制单回合影响但允许多回合策略。设计明确的冷却与优先级规则，避免因单一强力法宝导致游戏失衡。
6. 法宝有生命值，作战时会在人物前面，和敌人进行战斗，如果被打败，会进入损毁状态，需要修复，损毁状态无法进入战斗。
7. 每个法宝有对应的法术，法术有对应的法术图标。
8. 强大的法宝占用丹田更多，弱小的法宝占用丹田更少，多个弱小的法宝，作战效果不一定比一个强大的法宝弱。
9. “法宝进化”：
    1. 弱小法宝在温养到一定时间后，可以进化为更强大的法宝。
    2. 进化时形状会改变，可能更难拼合，但属性更强。
10. 法宝强化时，随机消除最外层一格，让形状变小，随着境界的增加，装备可强化的次数也随之增加。
11. “温养时间”机制
    1. 法宝在丹田温养时间越久，温养等级越高，属性越强。
12. 法宝羁绊：
    1. 法宝也可以分为不同类别，形成羁绊效果：
        剑类法宝（飞剑、剑阵）：数量达到一定时，触发群攻或连击。
        盾类法宝（玄龟盾、金钟罩）：数量达到一定时，提升防御或格挡率。
        火类法宝（火焰珠、赤焰刀）：数量达到一定时，附加灼烧效果。
        水类法宝（寒冰镜、碧水环）：数量达到一定时，附加减速或冰冻效果。
        辅助类法宝（灵灯、阵旗）：数量达到一定时，提升全队属性或提供治疗。
13. 法宝稀有度等级与属性倍率表(P0实现稀有度就好，具体功能后面再实现)：
稀有度	稀有度简称	生命值倍率	技能威力倍率	温养效率倍率	出场点数/体积影响

精良	Fine	1.10×	1.05×	1.05×	无变化

14. 法宝境界(P0实现境界就好，具体功能后面再实现)：
凡器，可装备入丹田，基础生命与技能，无法温养进化； 
灵器，可温养；解锁基础温养属性加成；
真器，解锁第二阶技能槽；温养效率提升；
法器，解锁器灵，解锁稀有度强化功能；
天器，解锁被动强化与形态变更；
圣器，解锁专属大招或本命技；
仙器，形态独特，属性大幅提升；
极道帝兵，最终形态，单独设计最终效果；


## 人物加点系统(P0)
1. 灵力（Spirit Power）
    1. 影响：丹田容量、法宝最大出场点数、法宝出场点数恢复速率、法术威力
    2. 加点效果：扩大丹田空间，提升法宝温养速度
2. 神识（Divine Sense）
    1. 影响：识海容量、功法数量上限、功法羁绊属性提升率
    2. 加点效果：扩大识海空间，提升功法理解度
3. 体魄（Body Constitution）
    1. 影响：角色生命值、防御力、法宝耐久度
    2. 加点效果：提升法宝生命值，增加角色抗性
4. 悟性（Comprehension）
    1. 影响：功法升级速度、功法羁绊属性提升率、特殊功法领悟
    2. 加点效果：加快功法温养，提升功法羁绊效果
5. 气运（Fortune）
    1. 影响：法宝强化成功率、功法掉落概率、特殊事件触发
    2. 加点效果：提升法宝进化几率，增加稀有功法获取率

## 战斗系统(P0)
1. 召唤规则：
    1. 每回合开始恢复行动点
    2. 召唤法宝消耗对应的行动点
    3. 可同时召唤多个法宝，只要行动点足够
    4. 法宝被击败后进入冷却，冷却结束后可再次召唤
2. 战斗流程：
    1. 回合开始 → 恢复行动点
    2. 敌人召唤阶段
        ├─ 敌人召唤法宝（敌人优先召唤，方便玩家根据敌人的法宝做出部署策略）
        └─ 无法宝时跳过召唤阶段
    3. 玩家召唤阶段
        ├─ 选择召唤法宝（消耗行动点）
        └─ 无法宝时跳过召唤阶段
    4. 玩家战斗阶段准备
        ├─ 已经召唤的法宝需要设置使用的技能、普通攻击、防御
        └─ 玩家可直接施放自身的法术
    5. 敌人战斗阶段准备
        ├─ 已经召唤的法宝需要设置使用的技能、普通攻击、防御
        └─ 敌人可直接施放自身的法术
    6. 战斗触发阶段
        ├─ 点击开始战斗按钮
        ├─ 玩家和敌人同时施放法术
    7. 结算阶段
        ├─ 检查法宝生命值
        ├─ 更新冷却状态
        ├─ 判断胜负
        └─ 结算奖励


## 境界系统(P1)
1. 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 飞升成仙
2. 每个境界解锁不同的系统，只解锁，强化只能靠加点(P0先实现到金丹期)：
    炼气期	基础战斗系统	玩家只能使用基础功法，无法召唤法宝；识海和丹田未开启
    筑基期	识海开启	可存放功法，开始触发功法羁绊；识海初始容量有限，需加点扩展
    金丹期	丹田栏开启	可存放法宝，开始召唤法宝参战；丹田初始容量固定，需加点扩展
    元婴期	法宝温养机制	法宝开始受温养时间影响，属性随时间提升；解锁法宝强化功能
    化神期	功法进阶	功法可进化为更高阶版本，羁绊效果增强；识海可容纳双系功法
    炼虚期	法宝进化	弱小法宝可进化为更强大的形态，形状改变；解锁法宝组合技
    合体期	功法与法宝联动	功法羁绊可影响法宝召唤效果，例如剑修功法增强剑类法宝
    大乘期	阵法系统	玩家可布置阵法，影响全队属性；阵法与功法、法宝产生三系联动
    渡劫期	天劫试炼	境界突破需经历战斗试炼；成功后解锁特殊被动技能
    飞升期	全系统融合	功法、法宝、阵法全面融合，解锁最终形态玩法（如本命法宝+功法合一）

## 功法系统(P1)
1. 功法存放识海里。
2. 功法配置占人物的功法点数，会影响修行的速度。
3. 功法有单独的识海栏，筑基期解锁识海栏后，识海栏只能靠加点增加。
4. 功法派系之间可以存在羁绊增益，可以通过build功法搭配来强化自身。
5. 功法例子：御剑诀、万剑归宗、剑心通明
6. 功法羁绊：
    1. 2本剑修功法：攻击速度提升10%
    2. 4本剑修功法：剑气范围攻击，伤害溅射30%
    3. 6本剑修功法：每次释放剑诀时额外触发一次剑影攻击

## 货币系统(P1)
1. 灵石的使用：
    1. 境界突破
    2. 购买物品
    3. 法宝强化
    4. 功法进阶


## 地图系统(P1)
1. 每个境界解锁不同的地图
2. 每个地图有固定的BOSS
3. 每个地图有固定的掉落

## 任务系统(P2)
1. 任务解锁地图
2. 任务解锁破镜丹，可以突破境界
3. 任务解锁法宝强化，中期才可以强化法宝
4. 任务解锁功法进阶，中期才可以进阶功法
5. 任务解锁一些材料的购买，未完成任务前只能通过打怪掉落

## 合成系统(P2)
1. 中期获得合成魔方道具，可以合成法宝。
2. 两个法宝合成一个，会只保留一个法宝的形状，属性词条概率性继承。
3. 增加一些辅助合成的道具，可以指向性合成形状，指向性继承词条。

     
## 彩蛋系统(P2)
1. 合成魔方可以有彩蛋
2. 剧情也可以有彩蛋
3. 像素动画可以有彩蛋
4. 功法冲突时可以有彩蛋buff


## 种田系统(P2)
1. 收集材料和种子，种植药田，获得收益，后续药材可以炼丹。
2. 升级需要灵石，药田开启种植后也会消耗药田的灵气，需要定时补充灵石。
3. 种子固定区域固定的怪可以爆出，收获时概率收获种子。


## 生活技能系统(P2)
1. 种植技能
2. 炼丹技能
