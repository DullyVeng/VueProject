# 游戏素材资源

由于 AI 图片生成服务暂时繁忙，以下为您推荐一些高质量的免费像素游戏素材网站，您可以直接下载使用：

## 推荐网站

### 1. itch.io (Game Assets)
全球最大的独立游戏资源库，有大量免费（Free）素材。
- **链接**: [https://itch.io/game-assets/free/tag-pixel-art](https://itch.io/game-assets/free/tag-pixel-art)
- **推荐关键词**: "RPG", "Tileset", "UI", "Icons"

### 2. OpenGameArt.org
老牌开源游戏素材网站。
- **链接**: [https://opengameart.org/](https://opengameart.org/)

### 3. Kenny Assets
质量非常高且完全免费（CC0）的素材包。
- **链接**: [https://kenney.nl/assets](https://kenney.nl/assets)

## 建议下载内容

1. **地图背景 (Map Background)**
   - 寻找一张 16:9 或更大分辨率的 "Pixel Art Landscape" 或 "RPG Village" 图片。
   - 保存至: `src/assets/game/images/maps/starter_village.jpg`

2. **UI 图标 (Icons)**
   - 寻找 RPG 图标包（包含剑、药水、盔甲等）。
   - 推荐 32x32 或 64x64 大小。
   - 保存至: `src/assets/game/images/ui/`
