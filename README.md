# 🌟 灵境 • 问道 (Xianxia Cultivation RPG)

## 📖 项目主要介绍

本项目是一款基于现代 Web 前端技术栈构建的**沉浸式修仙 RPG 网页游戏**。它不仅仅是一个游戏 demo，更是一次将传统修仙文化与现代游戏开发技术、以及 AI 辅助编程深度结合的创新实践。通过极简的浏览器入口，为玩家提供一个逻辑严密、策略深度、且极具成长感的修仙世界。

### 🛠️ 实用性 (Practicality)
* **完整的游戏闭环**：实现了从“练气修行”到“境界突破”，从“地图探索”到“回合制战斗”的完整核心玩法循环。玩家可以体验真实的数值成长反馈。
* **优秀的用户体验**：基于 **Vue 3** 的响应式设计，提供了丝滑的 UI 交互体验。无论是战斗中的即时反馈，还是复杂的背包/丹田管理，都能做到无卡顿、零延迟。
* **随时随地的娱乐**：作为 Web 应用，用户无需下载安装包，打开浏览器即可游玩。结合 **Pinia 持久化存储**，确保进度实时保存，完美适配碎片化时间。

### 💡 创意性 (Creativity)
* **“丹田空间”策略玩法**：创新性地设计了可视化“丹田”系统。法宝不再是枯燥的列表，而是具有不同“体积”和“形状”的实体。玩家需要在有限的丹田空间内进行**空间规划**（类似于背包整理玩法），以最大化战斗收益，实现了“纳须弥于芥子”的修仙设定。
* **五行生克与羁绊**：法宝与功法之间存在深度的羁绊系统。不同属性的法宝组合会触发特定的阵法特效（如剑阵、护盾），不仅还原了修仙小说的设定，更极大地增强了战斗的策略深度。
* **动态生成的探索**：地图探索引入了 Roguelike 元素，精英怪、宝箱、奇遇事件随机分布，让每一次的探索都充满未知的惊喜。

### ⚙️ 技术深度 (Technical Depth)
* **前沿技术栈架构**：
    * **Core**: 采用 **Vue 3 (Composition API)** + **Vite** 构建高性能前端应用。
    * **State**: 使用 **Pinia** 进行复杂的全局状态管理，完美处理角色属性、战斗计算、任务队列等多数据流的同步。
* **AI Agent 深度协同开发**：
    * 本项目采用了先进的 **AI-in-the-Loop** 开发模式。项目不仅仅是人工编写的，而是由开发者与专用 AI Agent（基于 Google DeepMind 理念）深度协作完成。
    * 包含完整的 **Agent 工作流**（详见 `ai-agent/` 目录），通过 AI 辅助进行复杂逻辑拆解、文档生成及代码审查，展示了未来软件工程的高效范式。
* **高可扩展的系统设计**：
    * 采用了模块化设计思想，将“战斗计算”、“地图生成”、“任务驱动”解耦为独立模块，不仅易于维护，更为后续扩展“宗门系统”、“PVP 玩法”打下坚实基础。
