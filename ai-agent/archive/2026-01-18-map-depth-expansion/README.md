# 小地图深度扩展 (2026-01-18)

本次迭代专注于提升小地图的探索深度，增加了精英怪系统、地图BOSS系统以及宝箱系统。

## 主要变更

### 1. 怪物系统升级
- **精英怪**：引入了精英怪系统，具有特殊的前缀词条（如狂暴、坚韧），生命值和攻击力显著提升，奖励也更为丰厚。
- **地图 BOSS**：每个地图区域（迷雾森林、青石山、幽暗洞穴）现在都有一个固定的 BOSS。
  - BOSS 拥有巨大的体型和特殊的光环特效。
  - BOSS 击败后有 24 小时的刷新冷却时间。
  - 掉落包括大量灵石、稀有材料、更有概率掉落封印法宝和建筑图纸。

### 2. 宝箱系统
- **三种宝箱**：木制、铁制（改为工具箱图标 `🧰`）、黄金宝箱。
- **奖励机制**：点击宝箱会弹出精美的奖励结算界面，物品直接存入背包。
- **保底机制**：修复了空宝箱的问题，增加了灵石保底。

### 3. UI/UX 优化
- **小地图显示**：BOSS、精英怪和宝箱现在都在小地图上有清晰的标识。
- **方向指引**：在屏幕边缘添加了动态箭头，指示 BOSS 的方位和距离。
- **奖励弹窗**：移除了原生的 `alert` 弹窗，改为沉浸式的游戏内 UI。
- **一键召唤优化**：一键召唤不再需要多余的确认点击，流程更加顺畅。

## 文件变更清单
- `src/views/ExplorationMapView.vue`: 核心逻辑更新，集成了奖励弹窗。
- `src/stores/exploration.js`: 状态管理增强，支持宝箱掉落和BOSS状态。
- `src/views/CombatView.vue`: 优化了一键召唤流程。
- `src/data/`: 新增 `eliteMonsters.js`, `mapBosses.js`, `chests.js` 等配置文件。
